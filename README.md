# Calc

[![Run on Repl.it](https://repl.it/badge/github/wheelercj/Calc)](https://repl.it/github/wheelercj/Calc)

A command-line calculator.

## Features:
- Operators: PEMDAS, factorial, modulo, comparison
- Constants
- The ans variable
- User-defined variables
